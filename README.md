# CycleNestOrchestrator

CycleNestOrchestrator is a RESTful orchestration service developed for the  
**Service-Centric & Cloud Computing (COMP30231)** module.

The service exposes REST endpoints deployed on **Apache Tomcat 9** using  
**JAX-RS (Jersey)** and demonstrates external service orchestration, JSON
processing, and robust error handling.
